# Assets Directory

This directory contains visual media for the Sauron project documentation.

## Current Assets

### Screenshots
- `dashboard-overview.png` - Full dashboard view showing the interactive map with site markers
- `offline-panel.png` - Close-up of the offline sites panel showing red/yellow status indicators

### Demo
- `demo.gif` - Animated demonstration of the tool in action (clicking sites, viewing status changes, etc.)

## Adding Your Own Images

To add the actual image files:

1. Take screenshots or record a GIF of the Sauron dashboard
2. Name them according to the convention above
3. Place them in this directory
4. The README.md already references these files

### Recommended Image Specifications

**Screenshots (PNG format):**
- Resolution: 1920x1080 or higher
- Format: PNG for crisp text and UI elements
- Keep file size under 2MB if possible

**Demo GIF:**
- Resolution: 1280x720 recommended
- Frame rate: 10-15 fps for smooth playback
- Duration: 10-30 seconds to showcase key features
- Keep file size under 5MB if possible (consider using tools like ezgif.com to optimize)

### What to Capture

**dashboard-overview.png** should show:
- The full geospatial map with multiple site markers
- Color-coded status (green ✓, yellow ⚠, red ✗)
- The left-hand offline panel
- Overall dashboard layout

**offline-panel.png** should show:
- Close-up of the failure list
- Site details and status codes
- Grouping by status (red vs yellow)

**demo.gif** should demonstrate:
- Starting the dashboard
- Clicking on different site markers
- Popup information appearing
- The interactive nature of the map
- Zooming or panning (optional)
